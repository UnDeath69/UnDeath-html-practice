CSS positioning

There are 5 types of positioning

  • Static
  • Relative
  • Absolute
  • Fixed
  • Sticky
Relative positioning

Absolute positioning

Absolute positioning allows you to position an element relative to its nearest positioned ancestor (an ancestor with a position other than static). If there is no such ancestor, it will be positioned relative to the initial containing block (usually the viewport). You can use the top, bottom, left, and right properties to specify the exact position of the element.
